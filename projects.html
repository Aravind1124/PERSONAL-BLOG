Project 1

Restaurant Management System

The Restaurant Management System is done using Tkinter and aims to streamline and automate various tasks within a restaurant environment. The system provides an intuitive graphical interface for users to interact with, facilitating operations such as order management, menu updates, reservation handling, billing, and inventory management.

Project 2

Online Clothes Rental Store

The Online Cloth Rental Store project aims to provide users with a convenient platform to rent clothes online for various occasions such as parties, weddings, or casual outings. Built using Python's Tkinter library, the application offers a user-friendly interface for browsing through available clothing options, placing rental orders, managing bookings, and processing payments.
